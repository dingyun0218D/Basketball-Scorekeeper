# 🏀 篮球计分器项目总结

## 项目概述

这是一个功能完善的现代化篮球计分器应用程序，采用 React + TypeScript + Vite 技术栈开发，支持实时比赛记录、多人协作、球员数据管理、统计分析等功能。

## 🎯 核心功能

### 1. 实时计分系统
- ✅ 支持1分、2分、3分得分记录
- ✅ 即时更新比分显示
- ✅ 自动记录得分事件
- ✅ 球员个人得分统计

### 2. 精确计时管理
- ✅ 倒计时器实现（每节12分钟）
- ✅ 开始、暂停、继续、停止功能
- ✅ 自动节次管理（1-4节）
- ✅ 时间格式化显示（MM:SS）

### 3. 完整球员管理
- ✅ 球员信息录入（姓名、号码、位置）
- ✅ 详细数据统计（得分、篮板、助攻、抢断、盖帽、犯规）
- ✅ 投篮统计（命中率、三分、罚球、出手统计）
- ✅ 实时数据更新

### 4. 多人协作功能
- ✅ Firebase协作服务支持
- ✅ LeanCloud协作服务支持
- ✅ 会话管理（创建/加入会话）
- ✅ 实时数据同步
- ✅ 在线用户状态显示
- ✅ 服务切换功能

### 5. 队伍管理系统
- ✅ 双队伍管理
- ✅ 队伍犯规统计
- ✅ 暂停次数管理
- ✅ 自定义队伍名称和颜色

### 6. 数据管理
- ✅ 本地存储实现
- ✅ 自动保存比赛状态
- ✅ 历史记录管理
- ✅ 比赛事件日志

## 🏗️ 技术架构

### 前端技术栈
```
React 18.2.0          # 前端框架
TypeScript 5.2.2      # 类型系统
Vite 5.0.8            # 构建工具
Tailwind CSS 3.3.6    # 样式框架
Firebase 11.9.1       # 协作服务
LeanCloud Storage      # 协作服务
```

### 核心组件
- **计分板** - 显示比分、时间、快速操作
- **球员管理** - 添加/编辑球员，统计管理
- **协作管理** - 多服务协作功能
- **历史记录** - 比赛数据查看和管理
- **统计分析** - 详细的数据分析和展示

## 🎨 用户界面特色

### 设计原则
- **响应式设计** - 适配桌面、平板和移动设备
- **直观操作** - 大按钮、清晰标识
- **实时反馈** - 即时更新、状态提示
- **专业外观** - 类似电视转播的计分板风格

### 主要界面
1. **计分板页面** - 显示比分、时间、基本操作
2. **球员管理与统计** - 球员数据管理和统计分析
3. **历史记录** - 过往比赛回顾和数据查看

## 📊 数据结构

### 核心实体
- **Player** - 球员信息和统计数据
- **Team** - 队伍信息和球员列表
- **GameState** - 比赛状态和事件记录
- **GameEvent** - 比赛事件日志

### 协作数据
- **Session** - 协作会话信息
- **UserStatus** - 在线用户状态
- **SyncEvent** - 同步事件记录
